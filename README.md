Radionics

Radionics is a desktop application designed to simplify working with radio electronic components. It allows users to calculate and determine key characteristics of components such as resistors, capacitors, inductors, fuses, and IP protection levels.

The application is intended for engineers, students, and anyone working with electronics, especially in situations where quick calculations are needed without access to reference materials.

Features

1. Resistor calculations (SMD code ↔ resistance)

2. Capacitor calculations (code ↔ capacitance)

3. Inductor value determination from color markings

4. Fuse current and wire diameter calculations based on material

5. IP protection rating decoding and selection

6. Simple and intuitive graphical interface

7. Support for bidirectional calculations (value ↔ code)

Technologies:

- C#

- .NET (Windows Forms)

Radionics

Radionics — це десктопний додаток, створений для спрощення роботи з радіоелектронними компонентами. Він дозволяє обчислювати та визначати основні характеристики таких компонентів, як резистори, конденсатори, індуктивності, запобіжники та ступінь захисту IP.

Програма орієнтована на інженерів, студентів та всіх, хто працює з електронікою, особливо у випадках, коли потрібні швидкі розрахунки без використання довідників.

Функціонал

1. Розрахунок резисторів (SMD код ↔ опір)

2. Розрахунок конденсаторів (код ↔ ємність)

3. Визначення індуктивності за кольоровим маркуванням

4. Розрахунок сили струму та діаметра провідника для запобіжників

5. Розшифровка та визначення ступеня захисту IP

6. Простий та зрозумілий графічний інтерфейс

7. Підтримка двосторонніх розрахунків (значення ↔ код)

Технології:

- C#

- .NET (Windows Forms)
